# Política
# Este portal es divulgativo. No procesa datos personales, no tiene formularios,
# no almacena información de usuarios. Si detectas alguna vulnerabilidad:
#   - Comunícala de forma responsable al contacto anterior
#   - No divulgues públicamente antes de recibir respuesta
#   - Respeta un plazo razonable de 90 días para la corrección

# Reconocimiento
# Los investigadores que reporten vulnerabilidades reales de forma responsable
# serán reconocidos públicamente (con su consentimiento) en la sección
# de agradecimientos del sitio.
